# Install

## Public beta expectation

voice-cli intends to ship installable beta binaries for:
- macOS
- Windows
- Linux

## Current prototype state

At the current stage, the project is still a foundation scaffold and does not yet ship packaged binaries.

## Planned install flow

1. Download the beta binary for your platform.
2. Launch voice-cli.
3. Complete first-run setup.
4. Select a supported CLI.
5. Choose a local project.
6. Test voice settings and transcript behavior.
