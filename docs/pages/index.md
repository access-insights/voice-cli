# voice-cli

voice-cli is a voice-first desktop app for conversational access to agentic CLI workflows.

## What it does
- runs a supported CLI inside a local project
- captures terminal activity
- converts output into structured events
- speaks concise summaries
- exposes raw output, diffs, errors, and prompts on demand
