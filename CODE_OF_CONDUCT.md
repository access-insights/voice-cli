# Code of Conduct

## Our pledge

We are committed to a welcoming, inclusive, and respectful community.

## Our standards

Examples of positive behavior include:
- using welcoming and respectful language
- accepting constructive feedback
- prioritizing accessibility and inclusion
- showing empathy toward users and contributors

Examples of unacceptable behavior include:
- harassment or discrimination
- insulting or demeaning comments
- deliberate exclusionary behavior
- publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Reporting

Please report concerns through the repository security or maintainer contact path.
